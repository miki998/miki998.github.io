" collection: portfolio --- The project aims to realize the detection and recognition of cats and dogs. The product shall be able to a more or less continuous recognition of the object. In order to achieve a real time reliable detection , we will be making use of 23 layer convolution neural network YoloV3-tiny architecture with 416x416x3 input layer format plus a two nodes output layer (cat/dog), on top of which a multi-classifier haar object HaarCascade algorithm. Recognition comes with the Local Binary Pattern Histogram which we use with 8 neighbouring box setting. Aligning as pre-processing and video-wise we won’t keep recognizing and use detection tracker to follow mulitple pets. 2D Closest point will be used. At the end, we manage to get a functional real-time detection and recognition tracker but tested only on a limited dabase of pet.
