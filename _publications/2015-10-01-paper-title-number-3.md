The project aims to realize detection and recognition of cats and dogs. The product shall be almost real time recognition of the object. In order to achieve so, we will be making use of 23 layers of convolution so a YoloV3-tiny architecture with size 416 input images and two output nodes (for cat and dog), on top of which a multi-classifier haar object Haar-Cascade algorithm. Recognition comes with the Local Binary Pattern Histogram which we use with 8 neighbouring box settings.
